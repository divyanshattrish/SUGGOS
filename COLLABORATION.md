## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/divyanshattrish/SUGGOS.git
cd SUGGOS
```

### 2. Configure Git (First Time Only)
```bash
git config user.name "Your Name"
git config user.email "your.email@example.com"
```

### 3. Create Your Feature Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Changes & Commit
```bash
git add .
git commit -m "Descriptive message about your changes"
```

### 5. Push & Create Pull Request
```bash
git push origin feature/your-feature-name
```
Then create a PR on GitHub for review.

### 6. Stay Updated
```bash
git pull origin main
```

## Sync Strategy
- **Pull before starting work** to get the latest changes
- **Push frequently** to share your progress
- **Use feature branches** - don't commit directly to main
- **Create PRs** for code review before merging

## Branch Naming Convention
- `feature/description` - New features
- `fix/description` - Bug fixes
- `docs/description` - Documentation updates

## Communication
Discuss changes in PR comments before merging. Use descriptive commit messages so everyone knows what changed.
